# Quickstart Instructions

Get up and running with the application in 3 simple steps:

### 1. Requirements
Ensure you have Python installed. Install the dependencies:
```bash
pip install -r requirements.txt
```

### 2. Start the Server
Launch the FastAPI application server:
```bash
python main.py
```

### 3. Access the Dashboard
Open your preferred web browser and go to:
[http://127.0.0.1:8000](http://127.0.0.1:8000)

---

**Note:** If you want to re-train the machine learning model from scratch with new data, run `python run_ml_pipeline.py` before starting the server.
