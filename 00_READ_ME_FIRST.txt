=========================================================
WELCOME TO THE LEAD CONVERSION PREDICTION PLATFORM
=========================================================

This application uses Machine Learning to predict lead conversion probabilities and prioritize sales efforts. 

QUICK START INSTRUCTIONS:
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python main.py`
3. Open your browser to: http://127.0.0.1:8000
